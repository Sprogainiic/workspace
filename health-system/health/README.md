# Health Canonical Memory Layout

## Canonical

- `daily_logs/` — approved daily summaries
- `weekly_summary/` — approved weekly reviews
- `metrics/weight.md`
- `metrics/adherence.md`
- `metrics/fatigue.md`
- `metrics/training_load.md`

Only Health Director writes canonical files.

## Scratch inboxes

- `inbox/fitness/`
- `inbox/nutrition/`
- `inbox/behavior/`
- `inbox/analysis/`
- `inbox/chef/`

Specialists may write here. Health Director decides what gets promoted.
